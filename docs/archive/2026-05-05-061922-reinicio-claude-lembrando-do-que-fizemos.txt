  debugger, terminal integrado), e em modo complain só loga sem bloquear.

  VS Code já mantido pela comunidade, testado, e bem mais restritivo na prática:

  - Bloqueia ~/.ssh, ~/.gnupg, ~/.aws, ~/.config (exceto VS Code)
  - Restringe filesystem a ~/Documents, ~/Downloads, ~/projects
  - Filtra syscalls perigosas
  - Bloqueia capabilities de admin
  - Cria namespace isolado pra processos filhos (extensões)

  3. Configurar code para sempre rodar dentro do firejail
